CC_G5 Firmware Installer

MobiFlight Glass Cockpit G5 Display for Flight Simulation

Hardware
ESP32-S3-DevKitC-1
Flash Mode
QIO · 80 MHz · 8 MB
Framework
MobiFlight + Arduino
Display
4-inch LCD (LovyanGFX)

Before You Flash

  • Connect the ESP32-S3 via USB
  • Use Chrome or Edge (Web Serial required)
  • Hold BOOT button if prompted by the installer
  • Configure device in MobiFlight Connector after flashing

Requires Chrome or Edge with Web Serial API support.
